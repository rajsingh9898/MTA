# 🌍 MTA Planner – AI Powered Travel Planning Platform

**MTA Planner** is an AI-powered travel planning web application designed to help users create smart, personalized, and optimized trip itineraries. The platform simplifies travel planning by generating recommendations, managing budgets, and organizing daily routes efficiently.

🔗 **Live Website:** https://mytripadvisior.co.in/

---

## 🚀 Overview

MTA Planner helps travelers:

- Generate AI-based trip itineraries
- Discover attractions, restaurants, and hidden gems
- Optimize travel routes
- Plan trips according to budget
- Organize group travel easily
- Explore destinations across 190+ countries

The goal of this platform is to make travel planning faster, smarter, and stress-free.

---

## 🧠 Key Features

### ✨ AI-Powered Trip Creation
Generate customized itineraries based on travel preferences, destination, duration, and budget.

### 📍 Smart Recommendations
Get personalized suggestions for attractions, activities, restaurants, and experiences.

### 💰 Budget Planning
Estimate and manage trip expenses effectively.

### 🗺 Route Optimization
Optimized daily schedules to minimize travel time.

### 👨‍👩‍👧 Group Travel Support
Plan trips for families, couples, or large groups.

### 🌎 Global Destination Coverage
Travel information and suggestions for 190+ countries worldwide.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository


git clone https://github.com/rajsingh9898/MTA
cd project

##2️⃣ Install Dependencies
pnpm install


##3️⃣ Configure Environment Variables

Create a .env.local file and add:

OPENAI_API_KEY=your_openai_key
MONGODB_URI=your_database_uri
RESEND_API_KEY=your_resend_key
NEXTAUTH_SECRET=your_secret
Nodemailer Mail:
Nodemailer Pass:


##4️⃣ Run Development Server
pnpm dev

Open in browser:

http://localhost:3000
